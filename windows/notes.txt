- wsltty
	- To get copy-paste to work in tmux:
		- Create the %LOCALAPPDATA%/wsltty/home/%USERNAME%/.minttrc file and add: `AllowSetSelection=true` following which selecting text using the mouse will allow pasting in Windows (e.g. Chrome)
		- %USERNAME% should be the windows username
		- %LOCALAPPDATA% should be C:\Users\%USERNAME%\AppData\Local
	- To use emoji in the terminal
		- Download https://github.com/iamcal/emoji-data/archive/v4.0.3.zip
		- Extract folder img-google-64 into /c/Users/MujtabaAbidHasan/AppData/Roaming/wsltty/emojis
		- Rename folder to 'google' (remove hyphens)
		- Add the following two lines to /c/Users/MujtabaAbidHasan/AppData/Roaming/wsltty/config:
			```
			Charwidth=unicode
			Emojis=google
			```
